# Haolin Ye — Academic Homepage

A dependency-free academic homepage designed for GitHub Pages.

## Local preview

From this directory, run:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## GitHub Pages

For a GitHub user homepage, publish these files from the `main` branch of a
repository named `Leaf-hl.github.io`. GitHub Pages will serve the site at:

`https://leaf-hl.github.io/`

No build step or GitHub Actions workflow is required.

## Privacy

The original CV and unpublished paper files are intentionally ignored by Git.
The public page uses only selected, non-sensitive information.
